# Noisy Animals Kata, Javascript in the Browser Edition

1. Run index.html in your browser. The tests should all run and all be green.
2. Refactor!

This small exercise is designed to help thinking about and improve code. 
`noisy_animal.js` has some poor code in it. Can you spot and fix some problems?

See [the root README.md](../README.md) for detailed instructions and thoughts.
